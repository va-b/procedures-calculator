# Калькулятор процедур

## Установить зависимости
```
npm run getpg
```

### Запустить в режиме разработки
```
npm run serve
```

### Собрать фронтенд
```
npm run build
```

### Запустить сервер
```
npm run serve
```

### Собрать приложение и развернуть сервер на локальной машине
```
npm run localDeploy
```
